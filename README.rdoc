== README

You will need to get your access_token from cbs site so you can access the cbs api.  Go to developer.cbssports.com to get this.

load a rails console and run Player.load_data.  This loads the players and rankings from cbs.

Next you will run the rake task adp to get the average draft poistion numbers from http://fantasyfootballcalculator.com.  This will update your player in the db.
